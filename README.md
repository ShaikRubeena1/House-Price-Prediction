# House Price Prediction using Machine Learning

## Project Overview

This project predicts house prices based on property characteristics such as area, number of bedrooms, bathrooms, parking spaces, and furnishing status.

The model helps estimate property values and supports better decision-making in the real estate market.


## Business Problem

Accurately pricing a house is challenging for buyers, sellers, and real estate agencies.

Overpricing may reduce buyer interest, while underpricing can lead to financial losses.

This project uses Machine Learning to predict house prices based on historical housing data.
